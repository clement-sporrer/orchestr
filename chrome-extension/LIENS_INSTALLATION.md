# 🔗 Liens d'installation - Extension Chrome ORCHESTR

## 📥 Installation rapide

### Option 1 : Page d'installation HTML (Recommandé)
Ouvrez le fichier `INSTALLER.html` dans votre navigateur pour un guide visuel interactif :
```
chrome-extension/INSTALLER.html
```

### Option 2 : Guide détaillé Markdown
Consultez le guide complet dans :
```
chrome-extension/INSTALLATION.md
```

### Option 3 : Guide dans l'application
Une fois votre application déployée, accédez à :
```
https://votre-domaine.com/settings/extension
```

## 🚀 Liens directs

### Depuis GitHub
1. **Télécharger le dossier** :
   - Allez sur : https://github.com/clement-sporrer/orchestr
   - Cliquez sur "Code" → "Download ZIP"
   - Extrayez le fichier et ouvrez le dossier `chrome-extension`

2. **Ou cloner le dépôt** :
   ```bash
   git clone https://github.com/clement-sporrer/orchestr.git
   cd orchestr/chrome-extension
   ```

### Installation dans Chrome
1. Ouvrez Chrome et allez à : `chrome://extensions/`
2. Activez le "Mode développeur"
3. Cliquez sur "Charger l'extension non empaquetée"
4. Sélectionnez le dossier `chrome-extension`
