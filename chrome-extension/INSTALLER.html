ORCHESTR
Extension Chrome - Installation
📥 Installation rapide
Suivez ces étapes pour installer l'extension Chrome ORCHESTR en moins de 2 minutes.
1 Ouvrir Chrome Extensions

Ouvrez Chrome et allez à l'adresse suivante :

chrome://extensions/

Ou via le menu : ⋮ → Extensions → Gérer les extensions

2 Activer le Mode développeur

En haut à droite, activez le bouton "Mode développeur"

3 Charger l'extension

Cliquez sur "Charger l'extension non empaquetée"

Sélectionnez le dossier chrome-extension de ce projet

⚠️ Important : Sélectionnez le dossier chrome-extension, pas le fichier manifest.json
4 Configurer l'extension

1. Cliquez sur l'icône ORCHESTR dans la barre d'extensions

2. Entrez l'URL de votre API (ex: https://orchestr.vercel.app)

3. Entrez votre email de connexion ORCHESTR comme clé API

4. Cliquez sur "Enregistrer"

5 Utiliser l'extension

1. Allez sur un profil LinkedIn

2. Cliquez sur l'icône ORCHESTR

3. Sélectionnez une mission (optionnel)

4. Cliquez sur "Capturer ce profil"

Ouvrir Chrome Extensions
